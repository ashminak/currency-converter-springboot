# Currency Converter Application

A Spring Boot based Currency Converter application that converts currencies using a live exchange rate API and stores conversion history.

## Features

- Real-time currency conversion
- Conversion history tracking
- REST API architecture
- Spring Boot
- Spring Data JPA
- MySQL Database
- Exception Handling
- DTO Pattern
- Auditing

## Tech Stack

- Java 17
- Spring Boot
- Hibernate/JPA
- MySQL
- Maven

## API Endpoints
GET /api/currency/convert

GET /api/currency/history

GET /api/currency/convert

GET /api/currency/history
